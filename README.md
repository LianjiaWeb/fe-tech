# fe-tech 

应届生学习平台
## 背景

这届同学很优秀，学习意愿很高。

为了满足同学们学习的愿望，我们内部评审出目前团队所需技能的书籍列表以及技能点

### 技能清单

| 技能点  		   | 参考资料  		 | 目标          |
|:------------- |:---------------:| -------------:|
|html	| [w3school在线教程](https://www.w3school.com.cn/)<br/>《JavaScript DOM编程艺术》 | 了解html常用标签以及dom操作|

| chrome调试     |        |            |
